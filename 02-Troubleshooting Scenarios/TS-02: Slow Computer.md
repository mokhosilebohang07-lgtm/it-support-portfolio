# TS-02: Slow Computer Performance

## Scenario
A user reported that their computer was taking a long time to boot and applications were slow to open.

---

## Problem

System performance was significantly degraded.

---

## Symptoms

- Slow startup
- Applications freezing
- High CPU usage
- High memory consumption

---

## Investigation

### Step 1: Open Task Manager

Reviewed CPU, Memory and Disk utilization.

### Step 2: Review Startup Applications

Checked unnecessary startup programs.

---

## Tools Used

- Task Manager
- Startup Apps

---

## Root Cause

Multiple unnecessary startup applications were consuming resources.

---

## Solution

Disabled non-essential startup applications.

---

## Outcome

Boot time improved and system responsiveness increased.
