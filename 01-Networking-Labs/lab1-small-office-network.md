# Lab 1: Small Office Network

## Objective:
Build a basic office network using routers, switches, and PCs.

## Tools Used:
- Cisco Packet Tracer

## Steps:
1. Added 1 router
2. Added 1 switches
3. Connected 4 PCs
4. Assigned IP addresses

## Testing:
- Used ping command to test connectivity between devices

## Outcome:
All devices successfully communicated within the network.
